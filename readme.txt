== Installation ==

1. Download the plugin and unzip it.
2. Upload the folder /highlightjs-for-wordpress to your /wp-content/plugins/ folder.
3. Activate the plugin from your WordPress admin panel.
4. Installation finished.

== Frequently Asked Questions ==

= Why was this plugin created? =

Wanted the simplest include for hightlight.js that I could find. You could easily add this via functions or even the header or enqueue of your own theme, but this was mainly created for portability.

= Why did you release this plugin? =

Useful to me. May be useful to others.
